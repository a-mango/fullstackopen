## Useful commands

+ Install create-react-app: `npm i -g create-react-app`
+ Create react app: `npx create-react-app [app-name]`
+ Remove git repo: `rm -rf .git`
+ Reinstall node modules: `rm -rf node_modules/ && npm i`
